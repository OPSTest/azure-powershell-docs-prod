# Disconnect-AzureAD

## SYNOPSIS
Disconnects the current session from an Azure AD tenant

## SYNTAX

```
Disconnect-AzureAD [-WhatIf] [-Confirm]
```

## DESCRIPTION

## EXAMPLES

### Disconnects your PowerShell session from a tenant
