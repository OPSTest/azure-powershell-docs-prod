# Remove-AzureRMPlan

## SYNOPSIS
The Remove-AzureRMPlan cmddlet removes the specified plan.
The plan should not have any references to offers to get removed

## SYNTAX

```
Remove-AzureRMPlan -Name <String> -ResourceGroup <String> [-SubscriptionId <Guid>] [-AdminUri <Uri>]
 [-Token <String>] [-ApiVersion <String>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [-PipelineVariable <String>]
```

## DESCRIPTION

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
Remover-AzureRMPlan -Name ComputePlan
```

Description

-----------

The example removes a plan named ComputePlan
