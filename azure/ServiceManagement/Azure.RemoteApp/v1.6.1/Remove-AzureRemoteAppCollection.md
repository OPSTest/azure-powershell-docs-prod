# Remove-AzureRemoteAppCollection

## SYNOPSIS
Removes an azure_2 RemoteApp collection.

## SYNTAX

```
Remove-AzureRemoteAppCollection [-CollectionName] <String> [-Profile <AzureSMProfile>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
The **Remove-AzureRemoteAppCollection** cmdlet removes an azure_2 RemoteApp collection.
